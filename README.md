# Timeout Runner

静态网页端小游戏，基于浏览器断网页面的小恐龙玩法改进。

## 操作

- `Space`: 开始 / 重新开始
- `↑`: 跳跃
- `↓`: 下蹲
- `←`: 玩家向左移动
- `→`: 玩家向右移动
- `Distance` 左侧按钮：切换 English / 中文

## 玩法

- 自动向前奔跑，跑道速度由游戏时间逐渐提高。
- 左右键控制角色在跑道上横向移动，不再改变跑道速度。
- 地面障碍需要跳跃躲避。
- 高位低净空障碍需要下蹲通过。
- 金币不会与障碍物重叠。
- 金币每次只出现 1 到 2 枚。
- 金币以地面横向排列为主，少量金币出现在空中。

## 计分

- 距离每增加 1m 得 1 分。
- 金币每枚加 10 分。

## 运行

直接打开 `index.html`，或启动本地静态服务器：

```powershell
python -m http.server 8765 -d "D:\Master\P2026-006_网页小游戏_TimeoutRunner\03_CODE_代码"
```

然后访问：

```text
http://127.0.0.1:8765/
```
